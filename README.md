# Memorize - Flashcard Learning System

A complete offline flashcard application with spaced repetition and multiple-choice quizzes.

## Features

- **Spaced Repetition System** - Cards scheduled for review based on performance
- **Multiple Choice Quizzes** - Test knowledge with 4-option answers
- **Course Management** - Create multiple courses with decks of cards
- **Import/Export** - Import cards from CSV or Excel files
- **Progress Tracking** - Track new, learning, review, and mastered cards
- **Dark Theme UI** - Modern dark interface
- **Statistics** - View learning progress

## Installation

```bash
pip install -r requirements.txt
python memorize.py
